# 步进电机双轴云台跟踪系统 — 技术报告

## 1. 系统概述

本系统基于 TI MSPM0G3507 微控制器，实现了一套双轴步进电机云台的闭环视觉跟踪方案。视觉传感器通过串口实时传回目标坐标，经自适应卡尔曼滤波后，由 PID 控制器驱动 X/Y 轴步进电机进行精确跟踪。

## 2. 算法设计与实现

### 2.1 PID 控制器的物理建模

系统的核心任务是消除图像中心与目标坐标之间的偏差 $e(t)$。

- **比例项 (P)**：误差越大，给电机的频率越高，提供主要的纠正力。但纯比例控制存在死区停滞或极限振荡的问题。
- **积分项 (I)**：用于克服静摩擦力，持续累积推力直到突破摩擦阈值。对 X 轴重负载场景尤为关键。
- **微分项 (D)**：捕捉误差变化率，在云台高速接近目标时产生阻尼力，抑制超调。

### 2.2 DDS 直接数字频率合成

步进电机驱动需要稳定的脉冲频率。若在主循环中通过软件延时翻转 IO，会因其他任务（串口解析、OLED 刷新）的耗时波动导致脉冲抖动甚至丢步。

**解决方案**：设计固定 10kHz 的硬件定时器中断作为时基，采用 16 位相位累加器。每次中断累加一个与目标频率成正比的增量，溢出时输出一个脉冲。这种方案使 CPU 负载恒定，频率调整平滑，避免了频率突变带来的机械冲击。

### 2.3 自适应卡尔曼滤波

在标准一维卡尔曼滤波基础上，增加了基于新息序列的 R 自适应调整：

- 每次滤波更新后，将新息存入滑动窗口缓冲区
- 计算窗口内新息的均方值作为当前测量噪声的在线估计
- 对估计值进行限幅处理，防止极端偏差

当测量噪声突然增大时，R 自动增大，滤波器更依赖预测值；噪声减小时则加快跟随测量值。

## 3. 调试过程中的关键问题与解决方案

### 3.1 X 轴静摩擦力与 Stick-Slip 效应

**问题**：X 轴承载 Y 轴电机和传感器，机械惯量大。在微小误差区内，PID 输出的低频脉冲不足以驱动电机，积分项持续累积后突然爆发，电机跳过目标点。

**解决方案 — 分段速度包络**：
1. **远场区**（>35px）：线性大功率输出，追求响应速度，最高允许 250Hz
2. **中场区**（18-35px）：限速至 100Hz，强制系统减速
3. **近场区**（<18px）：限速至 60Hz 匀速滑行，防止超调
4. **起步保底频率**：最小输出 40Hz，确保能克服静摩擦力推动沉重的 X 轴

### 3.2 视觉噪声过滤 — 追击/锁定状态机

**问题**：目标被遮挡或光线闪烁时，视觉传感器输出毛刺和跳点，导致云台误动作。

**解决方案 — 双状态机 + 时间防抖**：
- **追击态 → 锁定态**：当误差进入极小区域（X轴 1.5px / Y轴 1.0px）后立即锁定，清空 PID 积分
- **锁定态 → 追击态**：必须连续 3 帧数据都超出逃逸阈值（6.5px）才解除锁定
- 设计思路：在复杂视觉环境中，"不动"的系统稳健性远优于对噪声的即时响应

### 3.3 X 轴定时器初始化死锁

**问题**：将 Y 轴代码移植到 X 轴时，程序在 TIMG12 初始化阶段卡死。

**原因分析**：Y 轴使用的 Timer0（高级定时器）与 X 轴使用的 TIMG12（通用定时器）的 API 和寄存器偏移不同。在未开启 TIMG12 电源域的情况下直接写入寄存器，触发总线超时挂起。

**解决方案**：
1. 建立严格的外设电源启动序列：`enablePower` → `delay_cycles` → 配置寄存器
2. 放弃"关闭定时器来停止电机"的方案，改为"维持时钟运行、仅关闭相位步进"的软停止策略

## 4. PID 自动整定 — Twiddle 算法

### 4.1 实现方式

采用 Twiddle（坐标上升法）对 Y 轴 PID 参数进行离线自动优化。算法通过逐一扰动 Kp/Ki/Kd 参数并评估阶跃响应误差来搜索最优解。

### 4.2 遇到的问题

- **伪最优解**：算法曾收敛到一组几乎为零的参数——因为电机没动，自然没有超调，误差积分反而"好看"。需要在代价函数中引入位移有效性约束。
- **跨轮次状态污染**：上一组参数导致电机失控后位置偏移，影响下一轮测试的基准。解决方法是每轮测试后执行物理位置复位和 PID 状态清零。

### 4.3 实际效果

Twiddle 给出的参数作为性能上限参考。由于视觉反馈链路存在 30ms+ 的延迟（Twiddle 在 10kHz 物理反馈下调优），高增益参数在实际跟踪中会引发振荡。最终取 Twiddle 结果的 50%-70% 作为实际运行参数，以换取系统鲁棒性。

## 5. 后续改进方向

- **前馈控制**：提取卡尔曼滤波中的速度分量，叠加在控制输出上实现预判位移
- **非线性 PID**：针对不同方向和角度设计非线性增益查找表
- **静摩擦补偿**：将静摩擦力常数硬编码进控制器，使系统在微小波动下也能越过静摩擦死区
